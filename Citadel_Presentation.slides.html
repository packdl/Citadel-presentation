""" 
    Let's see how many members are in our CSV file 
    and the most used email domain and least used domain.
"""
import csv
from collections import Counter

c = Counter()
with open('members.csv','r') as FILE:
    reader = csv.DictReader(FILE)
    members = list(reader)
    print(f"There are {len(members)} total members.")
    for member in members:
        domain = member["Email Address"].split("@")[-1]
        c[domain]+=1
m_c_domains = c.most_common()
mcd, lcd = m_c_domains[0], m_c_domains[-1]
print(f"The most common domain is {mcd[0]} with {mcd[1]} member")
print(f"The least common domain is {lcd[0]} with {lcd[1]} member")

There are 4 total members.
The most common domain is example.com with 3 member
The least common domain is gmail.com with 1 member

Python for Microcontrollers¶

  1. https://micropython.org/
  2. https://circuitpython.org/

Thank Microsoft for UF2. Flashing your microcontrollers is now pretty easy to do.

Both MicroPython and CircuitPython are Python 3 variants.
Common Traits

  • Fewer batteries included
  • Both are closer to Python 3.4 than Python 3.11

Differences

  • MicroPython has some more adanced features like threading
  • CircuitPython is focused on ease of learning

Python & Raspberry Pi¶

The Raspberry Pi Foundation has both Single Board Computers (SBC) and microcontrollers

SBCs

  • Raspberry Pi 1, 3, 4
  • Raspberry Pi 400
  • Raspberry Pi Zero, Zero W, Zero 2 W

Microcontrollers

  • RP2040
  • Raspberry Pi Pico, Pico W

Programming your device¶

  • Raspberry Pi 4 Model B
  • Raspberry Pi Pico
  • Adafruit Macropad

Programming your Raspberry PI 4 Model B¶

  • Supported IDEs: Thonny, Mu, VsCode
  • Python: CPython
  • Libraries: gpiozero, SMBus, Adafruit_Blinka-CircuitPython APIs

Programming your Raspberry PI Pico¶

  • Supported IDEs: Thonny, Mu, VSCode
  • Python: MicroPython or CircuitPython
  • Libraries: Various CircuitPython or Micropython libraries

Programming your Adafruit Macropad¶

  • Supported IDEs: Thonny, Mu, VSCode
  • Python: CircuitPython
  • Libraries: CircuitPython official and community bundle
